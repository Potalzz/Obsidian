  
**React에서 State는 컴포넌트 내에서 관리되는 데이터**
- 이 데이터는 컴포넌트의 생명주기 동안 변경될 수 있으며, 변경될 때마다 React는 해당 컴포넌트를 다시 렌더링하여 업데이트된 상태를 반영합니다.

**구문**
![[첨부 파일 소스/Image File 1/Pasted image 20240318181258.png]]
- state: 현재 상태 값을 가리키는 변수입니다.
- setState: 상태를 변경하는 함수입니다.
- initialState: 상태의 초기값을 나타냅니다. 컴포넌트가 처음 렌더링될 때 사용됩니다.
