# 세탁기 알림 설정 앱
### 아이디어
다른 성별의 세탁기 사용 여부는 알 수 없음

### 기획 의도
기숙사에 세탁기 부족으로 일일이 시간을 확인하기에 어려운 부분에서 착안하여,
세탁기가 가동중인지 여부를 알려주는 앱.

learndry라는 이름에 맞게, 각 분야마다 서로 이끌어주고 배워가면서 프로젝트를 진행하며
care & support 해주며 같이 성장해나간다.

**멤버 구성:** 테크3, 디자인2, PM1

### 사용 방법
**가입**
설치 후 초기에만 닉네임과 본인 호실을 입력하여 가입한다.
- `타인에게 닉네임 공개는 되지 않음`
- `각자 성별에 해당하는 현황만 확인할 수 있음`

**세탁기/건조기 현황 파악**
층을 체크하면 세탁기와 건조기가 가동중인지, 혹은 몇 분 남았는지 확인할 수 있다.

**세탁기/건조기 사용 등록**
세탁기/건조기를 가동하고 핸드폰으로 가동시간을 입력하게 되면 남은 시간과 예상 종료 시간을 확인할 수 있다.
`이전 사용자가 빨래를 가져가지 않았을 경우, 알림 보내기 버튼을 눌러 익명으로 알림을 보낼 수 있다.`

**알림 발송**
가동이 종료되면 사용자에게 알림이 간다.
수거 버튼을 클릭 하지 않고, 세탁기가 가동중이 아닌 경우에는 10분 간격으로 알림이 발송된다.

### 가입 절차
**닉네임만 입력 하면 자동으로 로그인**
닉네임을 초기에 입력 후 목록에서 서칭하여 본인의 닉네임과 호실을 입력하여 가입.
`[닉네임,호실,성별]`형태로 데이터베이스에 미리 등록해놓는다.

- 장점: 가입이 1초만에 간단하다
- 단점: 타인이 앱 다운시 가입이 가능하다
