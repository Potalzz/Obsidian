Parameter(파라미터,매개변수)
- **Parameter는 모델 내부에서 결정되는 변수**, 데이터를 통해서 산출이 가능한 값
- 모델 내부에서 데이터를 통해 구해진다.
- 예측을 수행할 때, 모델에 의해 요구되어지는 값
- 측정되거나 데이터로부터 학습되어진다.
- 학습된 모델의 일부로 저장되어진다.
- ex. 인공신경망에서의 가중치 계수(weight coefficient), SVM(서포트 벡터 머신)에서의 서포트 벡터, 선형회귀나 로지스틱회귀분석에서의 결정계수, 편향(bias)
