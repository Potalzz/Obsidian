## 박스 레벨 vs 인라인 레벨
### 박스 레벨 요소 (Box-level elements)
박스 레벨은 기본적으로 새로운 레벨에서 시작하면 전체 가로 너비를 차지한다.
> **특징**
> 1. 전체 너비 차지 : 부모 요소의 전체 가로 너비를 차지한다.
> 2. 새 줄에서 시작 : 각 박스 레벨 요소는 새로운 줄에서 시작한다.
> 3. 여백과 패딩 : 상하 여백(margin)과 패딩(padding)을 가질 수 있다.
> 4. 다른 블록 레벨 요소 포함 가능 : 다른 블록 레벨 요소나 인라인 요소를 포함할 수 있다.

> **예시**
> 1. `<div>`
> 2. `<p>`
> 3. `<h1>` to `<h6>`
> 4. `<ul>`, `<ol>`, `<li>`
> 5. `<section>`, `<article>`


### 인라인 레벨 요소 (Inline-level elements)
인라인 레벨은 텍스트의 일부분처럼 취급되며, 줄 바꿈 없이 연속적으로 표시된다.

> **특징**
> 1. 연속적으로 배치 : 같은 줄에 연속적으로 배치된다.
> 2. 너비만큼 차지 : 콘텐츠의 너비만큼만 공간을 차지한다.
> 3. 새 줄에서 시작하지 않는다 : 기본적으로 줄 바꿈이 일어나지 않음.
> 4. 상하 여백 제한적 : 상하 여백(margin)과 패딩(padding)은 제한적으로 적용된다.
> 5. 다른 인라인 레벨 요소 포함 가능 : 다른 인라인 레벨 요소나 텍스트를 포함할 수 있다.


> 예시
> 1. `<span>`
> 2. `<a>`
> 3. `<strong>`, `<em>`
> 4. `<img>`
> 5. `<b>`, `<i>`
> 6. `<u>`, `<small>`, `<mark>`

### 정리
**박스 레벨 요소**
- 새로운 줄에서 시작
- 전체 가로 너비를 차지
- 다른 박스 레벨 요소나 인라인 레벨 요소를 포함할 수 있음

**인라인 레벨 요소**
- 연속적으로 배치
- 콘텐츠의 너비만큼만 공간을 차지
- 줄 바꿈이 없음
- 다른 인라인 레벨 요소나 텍스트를 포함할 수 있음

## Display
display는 크게 3가지로 구성되어 있다. block, inline, inline-block
block : 한 줄 전체를 차지해서 한 줄에 요소 하나만 배치 가능.
inline : 크기에 맞게 영역을 차지해서 한 줄에 여러개 배치 가능, 대신 높낮이 조절 x
inline-block : inline속성에서 높낮이 크기 조절 가능

## Flex Box
**Flex box**
Flex box란? `display:flex`
인라인 레벨로 생성하려면 `display : inline-flex`

Flex박스의 기본적인 축 개념

컨테이너를 display:flex 로 설정하면 flex박스가 되고, 기본적으로 좌,우의 메인축과 상,하의 교차축을 가진다.


***메인축***
- **flex-direction** : **메인 축 방향 설정**
	기본값으로(row)좌->우 방향이며 메인축의 방향을 바꾸거나(row-reverse) 컨텐츠를 교차축에 맞춰 정렬(column)할 수 있다.
- **justify-content** : **메인 축 아이템 정렬 방식**
	메인축의 아이템 정렬 방식을 결정한다. 왼쪽부터 정렬할지, 오른쪽부터 정렬할지, 중앙 정렬할지, 아이템 사이 여백을 만들어 줄지 설정.
- **flex-wrap** : **메인 축 아이템 여러 줄로 배치할 지 설정**
	메인 축 한 줄에 아이템이 꽉차면 다음 줄을 만들어서 아이템들을 나열해줌. wrap이 없으면, 한 라인에 컨텐츠가 압축되어 담기지만 wrap이 설정되면 컨텐츠가 원래 사이즈에 맞게 배치되고, 다음 행이나 열로 옮기면서 정렬된다.
- **align-content** : **메인 축 공간 조정**
	메인 축이 여러 개일 경우, 메인 축 사이의 공간을 조정.

***교차축***
- **align-items** : **교차축 아이템 정렬 방식**
	교차축(메인 축과 교차하는 서브축의 개념)의 정렬 방식을 결정한다.
- **align-self** : **개별 요소 컨트롤**
	다른 속성들은 컨테이너 자체에 적용하는데에 비해, align-self는 개별요소에 적용해서 교차축에 맞춰 위치 조정 가능. (1:1 맞춤형 align-items라 생각하면 됨)

***요소 크기 설정***
- **flex-basis** : **아이템의 초기 크기 설정**
	메인 축에 따른 아이템의 크기를 px단위로 설정.
- **flex-grow** : **아이템이 남는 공간 차지하는 비율 설정**
	아이템이 한 줄에서 남는 공간을 얼마나 차지할지 비율로 설정한다. 단일 요소가 flex-grow:1 로 혼자일 경우 모두 차지하고, flex-grow 1과 2 총 두개가 있으면 1:2 비율로 차지한다.
- **flex-shirink** : **아이템의 줄어드는 비율 설정**
	컨테이너의 줄어듬 정도에 따라 다른 아이템들에 비해 **상대적으로** 얼마나 줄어들지 설정. 0이면 줄어들지 않고 숫자가 높을 수록 상대적으로 더욱 줄어든다.
- 모든 아이템들은 최소값,최대값을 설정하여 늘어남과 줄어듬의 정도를 제한할 수 있다. (min-width,max-width)
- **flex shortcut**
  flex요소 3가지들을 속기법으로 설정할 수 있다. 기본적인 순서는 **flex : flex-grow | flew-shirink | flex-basis**이고, 2개일 시 grow, shirink 순서이며 마지막 단위가 px일 경우 basis이다. 1개인 경우는 단위가 없으면 grow, px일 경우에는 basis.

**중첩 flexbox**
요소 안에 묶여있는 요소를 flexbox로 만들어서 해당 묶음을 flexbox로 만들어서 배치할 수 있다.

`Ex)`
`nav를 flexbox로 만들고 그 안에 ul을 flexbox로 한 번더 만듬`
