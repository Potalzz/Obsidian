### 변수 선언된 위치로 이동 (Jump to Definition)

변수나 함수의 **선언 위치로 이동**할 때:

#### 방법 1: 단축키 사용

- ⌘ + 클릭 (Command + Click)  
    → 선언된 위치로 바로 이동합니다.
    
#### 방법 2: Option 클릭

- ⌥ + 클릭 (Option + Click)  
    → Quick Look 팝업으로 미리 보기 가능 (실제 이동은 안 함)
    

#### 방법 3: 메뉴에서 이동

- **Control + Command + J**  
    → 선언된 곳으로 "편집기 화면을 이동"시킵니다.

### 변수 사용된 위치(Reference) 찾기

#### 방법 1: 마우스 우클릭 → Find References in Workspace

- 변수나 함수에서 **우클릭** → `Find References in Workspace` 선택
    

#### 방법 2: Shift + Command + F (전체 검색)

- `⇧ + ⌘ + F`  
    → 워크스페이스 전체에서 해당 이름으로 검색


**접혀있는 코드 한 번에 모두 접기/닫기**
- cmd + option + shift + <- / ->
