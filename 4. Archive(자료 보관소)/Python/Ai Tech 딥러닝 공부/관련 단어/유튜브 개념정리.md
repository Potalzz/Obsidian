# 편미분
편미분이란?
두개의 변수로 이루어진 함수를
F(x,y) = Yx^2
한번에 두개를 미분하기는 어려우니 하나씩 살펴보는 것을 편미분.

X에대한 변화율만 보면 x에 대한 편미분.

X로 편미분 할 때는 Y는 상수 취급 하고 미분.
-> yx^2 = 2xy
Y로 편미분 할때는 x는 상수 취급하고 미분
-> yx^2 = x^2 (y는 1이 되므로)
