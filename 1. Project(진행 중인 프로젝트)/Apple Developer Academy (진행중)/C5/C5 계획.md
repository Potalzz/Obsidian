## 🎯 14일 visionOS 그래픽스 입문 로드맵

> 하루 1~2시간 정도 학습 기준  
> **목표**: visionOS에서 3D 오브젝트를 만들고, 위치·회전·크기를 조작할 수 있는 수준

---

### Day 1 – 좌표계 이해하기

- **개념**
    
    - 2D vs 3D 좌표
        
    - 원점(0,0,0)과 x,y,z 축
        
    - 좌표로 점을 표현하는 법
        
- **실습**
    
    - 종이에 3D 좌표축 그려보기
        
    - `(1, 0, 0)`, `(0, 1, 0)`, `(0, 0, 1)`이 어디인지 표시
        
- **목표**
    
    - “위치”를 숫자 3개로 표현할 수 있다는 걸 이해
        

---

### **Day 2 – 벡터 기본**

- **개념**
    
    - 벡터 = 크기 + 방향
        
    - 벡터 vs 스칼라
        
    - 벡터 덧셈, 뺄셈
        
- **실습**
    
    - Swift Playgrounds에서 `SIMD3<Float>` 생성하고 더해보기
        
    - 예: `let v = SIMD3<Float>(1,2,3) + SIMD3<Float>(0,1,0)`
        
- **목표**
    
    - 벡터 연산이 “좌표를 움직이는 것”임을 직관적으로 이해
        

---

### **Day 3 – 벡터 연산 심화**

- **개념**
    
    - 벡터 길이(√(x²+y²+z²))
        
    - 단위 벡터(길이가 1인 벡터)
        
    - 내적(dot) → 각도 구하기
        
    - 외적(cross) → 수직 벡터 만들기
        
- **실습**
    
    - Swift에서 `simd_length`, `simd_normalize`, `simd_dot`, `simd_cross` 사용
        
- **목표**
    
    - 방향 계산의 기초 습득
        

---

### **Day 4 – 행렬 맛보기**

- **개념**
    
    - 행렬 = 숫자 표
        
    - 행렬 × 벡터 = 좌표 변환 규칙 적용
        
    - 3×3 회전/스케일, 4×4 이동 포함
        
- **실습**
    
    - 2×2 행렬 곱셈 손으로 계산
        
    - `simd_float4x4`로 단순 스케일 변환 적용
        
- **목표**
    
    - 행렬이 “규칙 저장 장치”임을 이해
        

---

### **Day 5 – 이동(Translation)**

- **개념**
    
    - 점을 특정 거리만큼 이동시키는 변환
        
    - 4×4 행렬 마지막 열의 (x, y, z)
        
- **실습**
    
    - `simd_float4x4(translation: SIMD3<Float>(tx, ty, tz))` 생성
        
    - 점 하나를 이동시켜보기
        
- **목표**
    
    - 이동 변환 행렬의 구조 이해
        

---

### **Day 6 – 회전(Rotation)**

- **개념**
    
    - 회전축(axis)과 각도(angle)
        
    - Y축 회전 예시
        
- **실습**
    
    - Swift에서 `simd_float4x4(rotation:around:)` 구현
        
    - 큐브 회전 애니메이션 만들기
        
- **목표**
    
    - 회전 행렬의 역할 직관적으로 이해
        

---

### **Day 7 – 스케일(Scale)**

- **개념**
    
    - 각 축 방향 크기 조절
        
    - 4×4 행렬 대각선 값이 크기
        
- **실습**
    
    - 큐브를 x축 2배, y축 0.5배로 조절
        
- **목표**
    
    - 스케일 변환 구조 이해
        

---

### **Day 8 – 변환 합성**

- **개념**
    
    - 이동 → 회전 → 스케일을 한 번에 적용
        
    - 행렬 곱셈 순서의 중요성
        
- **실습**
    
    - `M = T * R * S` 순서로 합성 후 적용
        
- **목표**
    
    - 한 번의 곱셈으로 여러 변환 가능함을 이해

## 💻 14일 로드맵 맞춘 RealityKit 코드 샘플

아래는 로드맵에 맞춰 **하루에 1개 씩 실행할 수 있는 최소 예제**를 만든 거야.  
모두 visionOS / RealityKit 환경에서 동작 가능.

---

### Day 1 – 좌표계

swift

복사편집

`let point = SIMD3<Float>(1, 2, 3) print("점의 위치:", point)`

---

### **Day 2 – 벡터 연산**

swift

복사편집

`let v1 = SIMD3<Float>(1, 0, 0) let v2 = SIMD3<Float>(0, 1, 0) let result = v1 + v2 print("결과:", result) // (1,1,0)`

---

### **Day 3 – 길이, 단위벡터**

swift

복사편집

`let v = SIMD3<Float>(3, 4, 0) print("길이:", simd_length(v)) // 5 print("단위벡터:", simd_normalize(v))`

---

### **Day 4 – 행렬 곱**

swift

복사편집

`var m = matrix_identity_float4x4 let v4 = SIMD4<Float>(1, 2, 3, 1) let transformed = m * v4 print(transformed)`

---

### **Day 5 – 이동**

swift

복사편집

`func translationMatrix(_ t: SIMD3<Float>) -> simd_float4x4 {     var m = matrix_identity_float4x4     m.columns.3 = SIMD4<Float>(t.x, t.y, t.z, 1)     return m }  let move = translationMatrix(SIMD3<Float>(1, 2, 3)) print(move)`

---

### **Day 6 – 회전**

swift

복사편집

`func rotationY(_ angle: Float) -> simd_float4x4 {     let c = cos(angle)     let s = sin(angle)     return simd_float4x4(         SIMD4<Float>( c, 0,  s, 0),         SIMD4<Float>( 0, 1,  0, 0),         SIMD4<Float>(-s, 0,  c, 0),         SIMD4<Float>( 0, 0,  0, 1)     ) }  let rot = rotationY(.pi/4) print(rot)`

---

### **Day 7 – 스케일**

swift

복사편집

`func scaleMatrix(_ s: SIMD3<Float>) -> simd_float4x4 {     return simd_float4x4(         SIMD4<Float>(s.x, 0,   0,   0),         SIMD4<Float>(0,   s.y, 0,   0),         SIMD4<Float>(0,   0,   s.z, 0),         SIMD4<Float>(0,   0,   0,   1)     ) }  let sc = scaleMatrix(SIMD3<Float>(2, 0.5, 1)) print(sc)`

---

### **Day 8 – 변환 합성**

swift

복사편집

`let t = translationMatrix(SIMD3<Float>(1, 0, 0)) let r = rotationY(.pi/2) let s = scaleMatrix(SIMD3<Float>(2, 2, 2)) let combined = t * r * s print(combined)`

---

### **Day 9 – RealityKit 큐브 생성**

swift

복사편집

`import RealityKit import SwiftUI  struct ContentView: View {     var body: some View {         RealityView { content in             let cube = ModelEntity(mesh: .generateBox(size: 0.1))             let anchor = AnchorEntity(world: [0, 0, -0.5])             anchor.addChild(cube)             content.add(anchor)         }     } }`

---

### **Day 10 – 변환 적용**

swift

복사편집

`cube.transform.translation = SIMD3<Float>(0, 0.2, -0.5) cube.transform.scale = SIMD3<Float>(2, 1, 1)`

---

### **Day 11 – 부모-자식 좌표계**

swift

복사편집

### **Day 12 – 카메라 조작**

### **Day 13 – 조명 추가**

### **Day 14 – 종합 예제**

swift

복사편집

`let cube = ModelEntity(mesh: .generateBox(size: 0.1)) cube.transform.translation = SIMD3<Float>(0, 0.1, -0.5) cube.transform.rotation = simd_quatf(angle: .pi/4, axis: [0, 1, 0]) cube.transform.scale = SIMD3<Float>(1.5, 1, 1)`
