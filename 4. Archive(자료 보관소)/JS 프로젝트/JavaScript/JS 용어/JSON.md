JSON (JavaScript Object Notation)은 데이터 교환을 위한 가볍고 읽기 쉬운 형식. 주로 웹 애플리케이션에서 서버와 클라이언트 간에 데이터를 전송하는 데 사용되며, 텍스트 기반 형식으로, 쉽게 작성하고 읽을 수 있으며, 많은 프로그래밍 언어에서 쉽게 파싱되고 생성될 수 있다.

### 특징
- **텍스트 형식 :** 텍스트 형식익이 때문에 사람이 읽고 쓰기 쉽다.
- **키-값 쌍 :** 키-값 쌍으로 데이터를 표현한다.
- **언어 독립적 :** 특정 프로그래밍 언어에 종속되지 않아, 각각의 프로그래밍 언어에서 JSON을 파싱하고 생성할 수 있다.
- **구조화된 데이터 표현 :** 배열, 객체 등과 같은 구조화된 데이터를 표현할 수 있다.

### JSON 구문
- 데이터는 키-값 쌍으로 구성됩니다.
- 데이터는 중괄호 `{}`로 묶어 객체로 표현합니다.
- 배열은 대괄호 `[]`로 묶어 표현합니다.
- 키는 문자열이어야 하며, 큰따옴표 `""`로 감싸야 합니다.
- 값은 문자열, 숫자, 객체, 배열, `true`, `false`, `null` 중 하나일 수 있습니다.

**예제**
```
{
  "name": "John Doe",
  "age": 30,
  "isStudent": false,
  "address": {
    "street": "123 Main St",
    "city": "Anytown",
    "state": "CA"
  },
  "courses": ["Math", "Science", "History"]
}
```
